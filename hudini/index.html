Hudini

A powerful utility library with UI components and tools built on top of PhaserWind. Create beautiful, responsive game interfaces with ease.

Explore Components View on GitHub

Key Features

🎨

Beautiful UI Components

Pre-built components for buttons, cards, progress bars, and more

📱

Responsive Design

Components that adapt to different screen sizes and orientations

⚡

High Performance

Optimized for 60fps gameplay with minimal overhead

🎯

Easy to Use

Simple API with TypeScript support and comprehensive documentation

🔧

Customizable

Extensive theming options and customization capabilities

🌪️

Built on PhaserWind

Leverages PhaserWind's theming system and design tokens

Hudini requires PhaserWind as a peer dependency. Make sure to install both packages.
