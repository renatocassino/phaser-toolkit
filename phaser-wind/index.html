Phaser Wind Overview

Phaser Wind

A theming system and utility library for Phaser 3 games with Tailwind-inspired design tokens. Create consistent, beautiful games with a powerful design system.

Explore Components View on GitHub

Key Features

🎨

Design Tokens

Consistent color, spacing, and typography tokens inspired by Tailwind CSS

🌪️

Theming System

Powerful theming system with light/dark mode support and custom themes

⚡

High Performance

Optimized for 60fps gameplay with minimal overhead and memory usage

🔧

Utility Functions

Helper functions for common game development tasks and calculations

📱

Responsive Design

Built-in responsive design utilities for different screen sizes

🎯

TypeScript Support

Full TypeScript support with comprehensive type definitions
